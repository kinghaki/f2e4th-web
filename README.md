# Week1 - The F2E 活動網站設計
# 感謝：設計師[KT](https://2022.thef2e.com/users/12061579703802991521)


啟動方式
```
$ yarn
```
```
$ yarn dev
``` 


### 版本
- yarn : 1.22.17

### 主要資料夾說明
- views : 網站中所有頁面
- components : 元件

### 使用技術
- Vue3(Vite)
- RWD (Support: 1280x720 768x1024 375x667)
- CSS Animation
- lottie-web(json檔產生動畫效果)


### 部屬
- netify
